# Easy AI 竞品分析报告

## 竞品概览

### LobsterAI (lobsterai.org)
- **开发商:** 网易有道
- **定位:** 面向大众的桌面AI Agent（"中国版OpenClaw"）
- **技术栈:** Electron 40 + React 18 + TypeScript + Tailwind CSS 3 + Redux Toolkit + SQLite
- **开源:** MIT License (github.com/netease-youdao/LobsterAI)

### Agent Runtime OS (多个竞品)
- **Agno AgentOS** (agno.com/agentos) — 最成熟的商业产品，Web dashboard
- **SmythOS** (smythos.com) — 可视化构建器 + SDK + CLI
- **AgentOS by Framers** (agentos.sh) — 最丰富的开源功能集

## 具体改进清单（按优先级）

### P0 — 必须做（不做就没法看）

1. **Landing/欢迎页重做** — 品牌感、产品名、tagline、引导流程
2. **对话界面打磨** — 头像/角色标识、时间戳分组、消息间距优化
3. **亮色/暗色主题切换** — 至少支持系统跟随
4. **文件上传** — 拖放 + 点击上传 + 剪贴板粘贴图片
5. **设置页重构** — 分组tab化、API Key连接测试、模型列表自动拉取
6. **会话管理增强** — 搜索、重命名、固定、时间分组
7. **空状态优化** — 每个页面都需要有意义的空状态引导
8. **错误处理升级** — 结构化错误提示 + 重试 + 帮助链接

### P1 — 应该做（竞品标配）

9. **系统提示词/角色设定** — 用户自定义system prompt
10. **对话分叉/重新生成** — 重新生成上一条回复
11. **消息编辑** — 编辑已发送消息并重新生成
12. **对话导出** — Markdown/JSON/图片导出
13. **快捷键系统** — Cmd+N新建、Cmd+K搜索等
14. **多语言** — 至少中英双语
15. **模型参数调节** — temperature、max_tokens滑块
16. **Token用量显示** — 每条消息的token计数

### P2 — 差异化（加分项）

17. **技能/插件系统** — 可扩展能力
18. **自动记忆提取** — 从对话中自动提取用户偏好
19. **定时任务** — 自然语言创建定时AI任务
20. **对话模板/预设** — 一键创建特定场景对话

---

## 总结

Easy AI当前处于"能跑"的阶段，距离"能用"还有很大差距，距离"好用"差距更大。
核心问题不是功能缺失，而是**已有功能的质量太低**——UI粗糙、交互简陋、缺乏打磨。

建议优先提升P0的8项，这些改完后产品至少能达到"可以demo"的水平。
